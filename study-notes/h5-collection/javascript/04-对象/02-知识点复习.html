1.JS的组成 BOM 浏览器对象模型 DOM 文档对象模型 ECMAScript 规定了JS的核心语法 2.等于和全等于 赋值 = 等于 == :只判断值 全等于 ===:判断值和类型 3.关系运算符 < > <= >= 4.逻辑运算符 逻辑非 ! ->取反->true | false 逻辑或 :表达式1 ||表达式2 表达式1为真,返回表达式1,否则返回表达2 逻辑与:表达式1 &&表达式2 表达式1为真,返回表达式2,否则返回表达1 5.值类型和引用类型 值类型 引用类型:对象类型 区别: 值类型存储时具体的数据 引用类型存储是地址,该地址指向存储数据的空间 6.值类型和引用类型赋值 赋值:默认会把右边存储的信息复制一份给左边 值类型的赋值:把右边存储的信息(具体的数据)复制一份给左边 引用类型的赋值:把右边存储的信息(指向具体数据的地址)复制一份给左边 7.值类型和引用类型作为函数的参数 函数的调用:默认会把实参赋值给形参 值类型作为函数的参数:修改形参不会影响实参 引用类型作为函数的参数:形参和实参共享同一份数据,互相有影响 8.in 01.遍历对象的key 02.判断某个对象是否存在指定的属性(属性 in 对象) 03.in操作数组的时候,操作的是索引,不是元素 9.delete 01.删除对象的属性 02.删除未使用var声明的变量 03.不能删除使用var声明的变量 04.可以删除直接定义在window上的属性 05.有返回值;true->删除成功 false->删除失败 10.调试工具 打开:F12(windows) command+option+i/j(Mac) js的断点 = 普通断点 + 条件断点 11.异常处理 try{//可能会出错的代码} catch(e){//如果上面的代码出错了会执行这个代码块中的代码} finally{//不管有没有出错,都会执行这个代码块中的代码} throw 错误信息(字符串 |对象) 12.面向对象和面向过程编程 都是解决问题的一种思路 面向过程关注的是解决一个问题的一个一个的步骤 面向对象关注是这个问题所需要的对象 13.面向对象的概念 对象:具体的事物 属性:定义在对象中变量-->静态的描述信息 方法:定义在对象中的函数-->动态的行为特征
