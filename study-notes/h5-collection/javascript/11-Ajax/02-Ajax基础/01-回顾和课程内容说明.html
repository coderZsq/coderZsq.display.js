[01] 网络相关的基础概念 客户端:用户使用的可以联网发送请求能够为客户提供服务的设备. 服务器:为客户端提供服务(数据)的设备. 请求:客户端向服务器索要数据过程. 响应:服务器把数据返回给客户端过程. URL: 统一资源定位符,找到唯一的服务器资源. 协议 + 地址 + 资源路径 http://www.baidu.com/logo.png 协议:规则(约定). HTTP:超文本传输协议. 作用:规定客户端和服务器端之间通信的规则(格式). 版本:0.9 -- 1.0 -- 1.1(正在使用:默认使用了持久连接) -- 2.0(最新) 特点:简单 | 灵活 | 明文传输 HTTPS: HTTP + SSL(安全套接字层) 网络通信模式: 请求-响应模式 HTTP报文:首部 + 主体 HTTP请求报文:请求首部(请求行 + 请求头:对客户端以及请求的描述信息) + 请求主体 HTTP响应报文:响应首部(状态行 + 响应头:对服务器以及响应的描述信息) + 响应主体 请求参数:提交给后台的数据. 请求方法:GET请求 + POST请求 [02] 表单发送请求 GET请求 请求路径 + 请求方法 + 参数(key-value) POST请求 文件上传
